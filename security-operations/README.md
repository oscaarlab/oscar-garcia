# 🔐 Security Operations

This section contains labs related to detection, analysis, and security event investigation.

The focus here is on building practical skills for identifying suspicious activity, analyzing logs, and understanding how threats are investigated in operational security environments.

---

### 🔧 What You’ll Find Here

- Log analysis
- Threat detection
- Event investigation
- MITRE ATT&CK mapping
- Detection-oriented workflows
- Security analysis with Python tooling

---

### Lab Focus

These labs are designed to simulate operational security tasks such as:

- reviewing logs for unusual behavior
- identifying indicators of suspicious activity
- mapping activity to known attack techniques
- supporting investigation with lightweight tooling

---

### Environment Context

The work in this section is aligned with **banking and regulated environments**, where monitoring, visibility, and timely detection are essential.

Labs are developed in **controlled environments** to safely reproduce scenarios and validate detection-oriented approaches.
